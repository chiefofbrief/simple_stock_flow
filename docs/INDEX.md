# Repository Index

LLM-powered stock analysis workflow combining discovery, screening, fundamental analysis, and sentiment monitoring.

## Directory Structure

```
simple_stock_flow/
├── docs/              # Documentation, indexes, and navigation
├── guidance/          # Analysis frameworks, prompts, and glossaries
├── sources/           # Reference material and investment theory
├── scripts/           # Data collection and orchestration scripts
├── data/              # Script outputs and analysis artifacts
└── SESSION_NOTES.md   # Active investigation log and decision tracking
```

## Subindexes

| Index | Coverage |
|-------|----------|
| `docs/index_scripts.md` | All Python scripts: master orchestrators, market scanners, ticker analyzers, shared utilities |
| `docs/index_guidance.md` | Workflow guidance files and analysis skill files |
| `docs/index_sources.md` | Reference materials: raw texts, summaries, insights |
| `docs/index_data.md` | Data outputs: directory structure, file naming conventions, formats |
