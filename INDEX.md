# Repository Index

LLM-powered stock analysis workflow combining discovery, screening, fundamental analysis, and sentiment monitoring.

## Directory Structure

```
simple_stock_flow/
├── scripts/           # Data collection and orchestration scripts
├── Guidance/          # High-level workflow and analysis guidance
├── skills/            # Analysis skill files (prompts for specific tasks)
├── Sources/           # Reference material and investment theory
├── data/              # Script outputs and analysis artifacts
├── setup/             # Configuration and dependency management
└── [indexes]          # Navigation files (this file and subindexes)
```

## Subindexes

| Index | Coverage |
|-------|----------|
| `index_scripts.md` | All Python scripts: master orchestrators, market scanners, ticker analyzers, shared utilities |
| `index_guidance.md` | Workflow guidance files and analysis skill files |
| `index_sources.md` | Reference materials: raw texts, summaries, insights |
| `index_data.md` | Data outputs: directory structure, file naming conventions, formats |
